Count of "http" and "ftp" occurrences in datasets

While the HTTPS protocol was always recommended for sites to use, the Google Chrome web browser has been blocking mixed content on websites. A mixed content refers to secured sites through the HTTPS protocol, but contains elements or resources that are hosted in a non-secure site (through HTTP or FTP). It is therefore highly recommended to update all HTTP and FTP links to an HTTPS alternative.
